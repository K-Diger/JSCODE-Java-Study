# 주의사항

JSCODE 1~3기가 시험을 봤다. 1, 2기는 평균 점수가 60점 이상이어야 합격이다.

3기는 평균 점수가 70점 이상이어야 합격이다.

다만, 100점 과목이 2개 이상일 경우 평균 점수와 상관없이 합격이다.

- 합격일 경우 `합격입니다.`라는 문구를 출력해야 한다.
- 불합격일 경우 `불합격입니다.`라는 문구를 출력해야 한다.
